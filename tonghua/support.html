🔧 Frequently Asked Questions

How do I set up API keys?
Go to Settings → API Settings, then enter your DeepSeek and MiniMax API keys. You can obtain these keys from their respective websites.
Why can't I generate stories?
Please check: 1) API keys are correctly configured 2) You have internet connection 3) Your API keys have sufficient credits
How does voice cloning work?
Record a 15-second voice sample, then the app will create a voice model. This requires a MiniMax API key and may take a few minutes to process.
How do I restore my purchase?
Go to Settings and tap "Restore Purchase". This will restore your premium features if you've previously purchased them.
Is my data safe?
Yes! All your stories and voice recordings are stored locally on your device. We don't collect or store personal information on our servers.

🔧 常见问题

如何设置API密钥？
进入设置 → API设置，然后输入您的DeepSeek和MiniMax API密钥。您可以从它们各自的官网获取这些密钥。
为什么无法生成故事？
请检查：1）API密钥配置正确 2）网络连接正常 3）API密钥有足够的额度
语音克隆如何工作？
录制15秒语音样本，应用会创建声音模型。这需要MiniMax API密钥，处理可能需要几分钟时间。
如何恢复购买？
进入设置，点击"恢复购买"。如果您之前购买过高级功能，这将恢复您的高级功能。
我的数据安全吗？
是的！所有故事和语音录音都存储在您设备本地。我们不会在服务器上收集或存储个人信息。
